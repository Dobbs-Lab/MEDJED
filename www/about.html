MEDJED

MEDJED (Microhomology Evoked Deletion Judication Elucidation) is a tool for assessing the probability of microhomology-mediated end joining repair at a particular DNA double-strand break site.

MEDJED is a random forest method trained using 5-fold cross validation on a set of 46 HeLa cell targets. It uses sequence features incorporating information about the length, GC content, distance between microhomology arms, and distance between microhomology arms and the DSB site to predict the proportion of deletion events due to MMEJ repair.


On an independent test set of 44 HeLa cell tragets, MEDJED achieves a mean absolute errror (MAE) of 14%, root mean square error (RMSE) of 16%, and Pearson Correlation Coefficient (CC) of 68% when comparing the predicted proportion of MMEJ repair outcomes with the observed proportion of MMEJ repair outcomes.

Download MEDJED

A standalone version of this code can be freely downloaded from our GitHub repository.

There are extensive installation and usage instructions available in the GitHub README file.

You can clone the repository with the following git command:

git clone https://github.com/Dobbs-Lab/MEDJED.git

The MEDJED R code is provided as-is; please be aware that you modify the code at your own risk. We are unable to provide technical support for modified versions.


Run MEDJED Locally

If you have R installed on your system, you can also follow the instructions here to easily run the MEDJED RShiny app from R, without dealing with Git.

MEDJED is also available as a Docker image. If you have Docker installed, you can clone the Docker image using the following command:

sudo docker pull cmmann/medjed
